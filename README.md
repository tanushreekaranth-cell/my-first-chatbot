# my-first-chatbot
a sample rule based chatbot developed using python
🤖 Python Chatbot (Text-Based)

📌 Overview

This is a simple text-based chatbot built using Python. It interacts with users by taking input and responding based on predefined keywords. The chatbot also includes features like dynamic responses, time/date display, and a help menu.

---

🚀 Features

- Keyword-based responses
- Dynamic replies using "random"
- Displays current time ⏰
- Displays current date 📅
- Help menu for user guidance
- Exit command to stop chatbot

---

🛠️ Technologies Used

- Python
- Random module
- Datetime module

---

▶️ How to Run

1. Copy the code into a file:

chatbot.py

2. Run the program:

python chatbot.py

3. Start chatting in the terminal

---

💬 Example

You: hello
Bot: Hi 😊

You: time
Bot: Current time is 14:30:22
